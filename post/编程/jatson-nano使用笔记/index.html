Jetson Nano 使用笔记

没有板载 wifi

默认ssh开启

没有vnc支持

相比于树莓派 开箱可用性更差， 需要更多Linux知识

Python 环境

统一使用 sudo （tensorflow在虚拟环境会出错）

1
2
3
4

sudo apt install python3-dev
sudo apt install python3-pip
sudo pip3 install virtualenv
# virtualenv -p /usr/bin/python3 --system-site-packages ~/pyenv3

jupyterlab 协作环境

1
2
3
4
5
6

# source ~/pyenv3/bin/activate
# sudo 安装 jupyterlab
sudo pip3 install jupyterlab
sudo pip3 install install jedi==0.17.2 # https://github.com/ipython/ipython/issues/12740
# 在sudo中启动 jupyterlab, 否则tensorflow/pandas都不能用，是因为硬件加速需要权限
sudo jupyter lab --collaborative --ip="*" --ServerApp.token=nano --allow-root
