前言

打算长期阅读「个人计算」相关的论文，对于一些特别喜欢的，会顺手做个翻译。今晚原本打算翻译 Kay - The Real Computer Revolution Hasn’t Happened Yet.pdf, 在 Google 一搜，发现ConanXin之前已经翻译过这篇文章了。为了避免做不必要的工作，晚上花了些时间搜索了下「个人计算」相关论文的中文翻译，把能找到的东西都放在 Github 的这个项目里: Personal-Computing-Papers

欢迎大家一起来维护。
