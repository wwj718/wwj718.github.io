[译]Alan Kay 在 OOPSLA'97 中提到的 Web 的替代解决方案是什么？

我不记得26年前说了什么…

我对 web 人士和浏览器人士显然没有注意过以前更好的前瞻性工作感到震惊，这可能会对事情的发展和现在产生重大影响。

三个重要的例子是

  1. 道格·恩格尔巴特的NLS系统（更重要的是：恩格尔巴特对协作和沟通的愿景）
  2. 苹果的Hypercard系统，它在80年代末期非常出色，并且更重要的是：展示了网络如何成熟以造福所有用户的路径
  3. Postscript如何解决了重要的系统问题。
