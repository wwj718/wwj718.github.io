前言

原文: Trygve M. H. Reenskaug Personal programming and the object computer

译文

摘要

我的目标, 是为那些想超越现成的应用程序, 并创建程序来控制他们的电子环境的普通人, 创造一种直观的计算机。我提交了 Loke，一种新型计算机，它是一个由对象组成的宇宙，一切皆是对象。我称它为对象计算机。Loke 采用 Squeak （Smalltalk 的一个变体） 实现，是一种用于执行、检查和探索的可扩展概念模型。它首先被用来演示, 一个新手( Ellen)如何通过一个适应她的能力、需求和偏好的 GUI 来编程一个智能闹钟。非正式的演示表明，非专业人士立即掌握了通信对象(communicating objects)的想法，这些对象代表了他们环境中的真实事物。他们还想将其用于自己的目的。他们在确定 Loke 的个人机遇和勾画出他们的实现方面很有创意。有趣的是，参加演示的专家程序员并没有看到 Loke 的意义。我已经完成了 Loke qua 概念模型的编程。该模型支持其潜在的安全性和隐私性，并维持其对象和消息模型。Loke qua 的编程环境仍处于起步阶段，其固有的安全和隐私属性在实践中仍未实现。未来的 Loke 设备将可以从任何地方访问，并嵌入到自己的硬件中来实现它们。Loke IDE 依赖于数据(Data)-上下文(Context)-交互(Interaction)（DCI），这是一种新的编程范式，可以带来具有清晰架构的可读代码。我提交 Loke 是为了享受个人编程的乐趣。
