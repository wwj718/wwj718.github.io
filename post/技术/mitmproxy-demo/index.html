使用mitmproxy查看红包照片

2016-01-26
技术

文章目录

#微信红包照片 今天下午，微信朋友圈有个限时活动，发红包看照片，一时人山人海，秀恩爱都秀出了朦胧感与新高度，单身狗们纷纷付费被虐，场面空前惨烈。

为响应动物保护协会号召，虽然我们做不到没有恩爱就没有伤害，但收费秀恩爱毕竟不太人道了，于是作为新世纪有爱心的码农，义不容辞地决定做些什么

#mitmproxy

mitmproxy is an interactive, SSL-capable man-in-the-middle proxy for HTTP with a console interface.

它可以用来拦截、修改、重放和保存 HTTP/HTTPS 请求。

#安装 pip install mitmproxy

#Usage 在命令行里输入mitmproxy -p 1234

不熟悉命令行的用户可以使用它的web ui界面: mitmweb -p 1234

使用ifconfig查看电脑的ip，之后将其填入手机代理(port为我们指定的1234)

然后打开你的朋友圈，点击你想看的照片,mitmproxy就能抓取到它的url了

将此url复制到浏览器，不需付费也能被秀一脸恩爱啦233

哦对了 如果你在mac下，你需要按住option键再拖动鼠标才能复制
