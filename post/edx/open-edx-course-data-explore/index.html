get_course_about_section

  • 方法路径： from courseware.courses import get_course_about_section

  • 源码: get_course_about_section

  • Valid keys:

    • overview
    • short_description
    • description
    • key_dates (includes start, end, exams, etc)
    • video
    • course_staff_short
    • course_staff_extended
    • requirements
    • syllabus
    • textbook
    • faq
    • effort
    • more_info
    • ocw_links

  • 案例:course_about.html
