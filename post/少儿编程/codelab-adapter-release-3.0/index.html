The big idea is messaging

这是目前的第三个大版本。

从CodeLab Adapter第一个原型到第三个版本，经历了2年有余的时间。

这期间，在软件工程上，Alan Kay和Bret对我产生了巨大影响。这个版本中留下了一些烙印。

这个版本是我自己相对满意的一个版本，花了很长时间深入messaging，过去在架构设计上痛苦纠结都得以缓解。

致敬伟大的Alan Kay。

彻底取消GUI，Web UI也是可选的

由于取消对任何GUI依赖，所以理论上可以运行在任何能够运行Python的地方。

增加对Arduino的支持

目前最流行的三大开源硬件是:

  • 树莓派
  • Arduino
  • micro:bit

近期我们也会将mrYsLab对ESP8266和树莓派GPIO的支持移植过来。

RoboMaster

RoboMaster EP横空出世之后，大家觉得在这个赛道上，一骑绝尘。

跟RoboMaster团队一直保持着连续，他们之前到访过CodeLab，与YY、ZQ也一直保持着沟通。

致敬

最后致敬Alan kay，所有这些变化之所以能够连贯地进行，而不至于坍塌，因为它本质上一个消息系统，拥有惊人的灵活性，这类系统在变化过程中，没不是在重塑某种结构。更恰当的隐喻是，它像生物，是可生长的。

这正是Alan Kay关于面向对象的核心见解，从未如此愉快地编程。
