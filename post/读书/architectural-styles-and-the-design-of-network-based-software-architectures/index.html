架构风格与基于网络的软件架构设计

2015-07-13
读书

文章目录

花了两天读Roy Thomas Fielding这篇著名的博士论文，度过了一个智商欠费的周末（其实获益匪浅啦）。最初翻的是英文版，由于英文渣渣的缘故，只好读中文版了（说好的坚持阅读英文文献呢orz）

Roy Thomas Fielding 有多屌呢，下边是维基百科的介绍

one of the principal authors of the HTTP specification, an authority on computer network architecture[2] and co-founder of the Apache HTTP Server project.

作者在绪论中提到他的动机是希望理解和评估基于网络应用软件的架构设计,通过有原则地使用架构约束,从而从架构中获得所期待的功能、性能和社交三方面的属性。

一种架构风格是一组已命名的、相互协作的架构约束。

每一个对REST架构感兴趣的人都值得亲自翻阅呢

—摘自Roy Thomas Fielding《架构风格与基于网络的软件架构设计》—
